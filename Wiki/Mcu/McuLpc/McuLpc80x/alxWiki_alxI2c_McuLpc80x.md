## Ctor Arguments
- __AlxI2c* me__ - Pointer to I2c object
- __I2C_Type* i2c__ - I2c peripheral base pointer
- __AlxIoPin* io_SCL__ - Pointer to SCL IoPin
- __AlxIoPin* io_SDA__ - Pointer to SDA IoPin
- __AlxI2c_Clk clk__ - I2c clock config enum
